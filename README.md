📚 Library Management System
------------------------------------------------------------------------------------------------------------------------------------------------------------------
📖 Overview

The Library Management System is a software application designed to manage books and users in a library efficiently.
It allows administrators to add, search, issue, and return books, making the library workflow organized and simple.

This project was developed as part of a learning exercise in Java and database integration.

------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 Features

📘 Add new books to the library

🔍 Search books by title or ID

📥 Issue books to students

📤 Return books

👤 Maintain student records

🗄 Store data using a database

------------------------------------------------------------------------------------------------------------------------------------------------------------------

🛠 Technologies Used
Technology	Purpose
Java	Backend logic
MySQL	Database
HTML/CSS	Frontend interface
JDBC	Database connectivity

------------------------------------------------------------------------------------------------------------------------------------------------------------------

🏗 System Architecture

The system follows a simple 3-layer structure:

Frontend

User interface for interacting with the system.

Backend

Java handles business logic and operations.

Database

MySQL stores book and student records.
